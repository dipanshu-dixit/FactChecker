Built for Truth

CrawlConda is a source-grounded verification system. Submit any claim and it searches 34 live news feeds, extracts only facts explicitly stated in sources, and issues a verdict — pinned permanently to IPFS.

Every result is auditable. Every source is cited. Human votes run alongside every AI verdict, forever.

The Pipeline

1
Expand — claim is decomposed into precision search keywords
2
Search — 34 RSS feeds + Google News scanned, top 8 articles ranked by relevance
3
Extract — facts pulled only from source text, no model hallucination permitted
4
Verify — verdict issued with exact headline citation and source URL

Verdicts

VerdictMeaning
CONFIRMEDSources explicitly report the event
PARTIALSources support part of the claim but not all details
UNCONFIRMEDSources found but none mention the claim
FALSESources explicitly contradict the claim

Discord Bot

Run fact-checks directly from Discord. All results sync to the web in real time.

1
Join the server — Click the Discord button below
2
Type the command — You MUST type !verify followed by your claim
3
Wait ~40 seconds — Bot will post verdict with sources
4
Vote with reactions — React 👍 or 👎 to cast your vote
⚠️ Important:
  • You MUST start with !verify command
  • Example: !verify Iran struck Qatar
  • Don't just type the claim without !verify
  • Wait for the bot to respond (takes ~40 seconds)

API Keys

1
Generate key — Go to API Docs and fill the form
2
Copy the key — Save it somewhere (shown only once!)
3
Save in Settings — Go to Settings, paste key, click Save
4
That's it! — Key is auto-loaded and used on all future requests
✅ No manual work needed:
  • Key is saved in your browser (localStorage)
  • Automatically attached to every verification request
  • No need to paste it anywhere else
  • View usage stats anytime in Settings

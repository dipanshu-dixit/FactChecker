Built for Truth

CrawlConda is a source-grounded verification system. Submit any claim and it searches 34 live news feeds, extracts only facts explicitly stated in sources, and issues a verdict — pinned permanently to IPFS.

Every result is auditable. Every source is cited. Human votes run alongside every AI verdict, forever.

The Pipeline

1
Expand — claim is decomposed into precision search keywords
2
Search — 34 RSS feeds + Google News scanned, top 8 articles ranked by relevance
3
Extract — facts pulled only from source text, no model hallucination permitted
4
Verify — verdict issued with exact headline citation and source URL

Verdicts

VerdictMeaning
CONFIRMEDSources explicitly report the event
PARTIALSources support part of the claim but not all details
UNCONFIRMEDSources found but none mention the claim
FALSESources explicitly contradict the claim

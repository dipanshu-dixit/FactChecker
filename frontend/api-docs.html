CrawlConda API

Programmatic access to the Ground Truth Engine

🚀 Getting Started

The CrawlConda API allows you to verify claims programmatically, retrieve past verdicts, and embed verification badges on your website.

Base URL

Or direct (if your network allows): https://factchecker-production-3945.up.railway.app

📡 API Endpoints

GET /verify FREE

Verify a claim by searching 34 live news sources and issuing a verdict.

Query Parameters

Parameter Type Description
claim string The claim to verify (min 8 characters)

Headers (Optional)

Authorization: Bearer YOUR_API_KEY

Example Request

curl "https://fact-checker-teal.vercel.app/api/proxy?path=/verify?claim=Iran%20struck%20Qatar" \
  -H "Authorization: Bearer cc_live_..."

Example Response

{
  "claim": "Iran struck Qatar gas facilities",
  "verdict": "CONFIRMED",
  "emoji": "✅",
  "summary": "Multiple sources confirm the attack...",
  "sources": [
    {
      "title": "Iran Attacks Qatar LNG Facility",
      "url": "https://...",
      "source": "BBC"
    }
  ],
  "ipfs_hash": "QmXxx...",
  "ipfs_url": "https://gateway.pinata.cloud/ipfs/QmXxx...",
  "timestamp": "2026-03-26T10:30:00Z"
}
GET /verdict/{ipfs_hash} FREE

Retrieve a specific verdict by its IPFS hash.

Example Request

curl "https://fact-checker-teal.vercel.app/api/proxy?path=/verdict/QmXxx..."
GET /verdicts FREE

List recent verdicts (up to 200).

Query Parameters

Parameter Type Default
limit integer 20 (max 200)
GET /trending FREE

Get top 5 most-voted verdicts in the last 24 hours.

POST /confirm/{ipfs_hash} FREE

Cast a vote (up/down) on a verdict.

Request Body

{
  "vote": "up",  // or "down"
  "user_id": "your_unique_id"
}
GET /badge/{ipfs_hash}.svg FREE

Get a "Verified by CrawlConda" SVG badge for embedding.

Example

<img src="https://fact-checker-teal.vercel.app/api/proxy?path=/badge/QmXxx.svg" alt="Verified" />
GET /badge/{ipfs_hash}/embed FREE

Get HTML and Markdown embed codes for a badge.

Response

{
  "badge_url": "https://.../badge/QmXxx.svg",
  "embed": {
    "html": "<a href=\"...\"><img src=\"...\" /></a>",
    "markdown": "[![Verified](...)](...)",
    "url": "https://fact-checker-teal.vercel.app/#/v/QmXxx"
  }
}
GET /api-keys/usage AUTH

Check your API key usage statistics.

Headers

Response

{

  "total_requests": 1337,

⚠️ Error Codes

Code Meaning
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
404 Not Found - Verdict doesn't exist
429 Rate Limit Exceeded - Wait or upgrade
503 Service Unavailable - Try again later
